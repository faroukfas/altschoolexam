ABOUT THE PROJECT

This project involves hosting a website on an Ubuntu server using Apache, configuring the server to serve the website's HTML and CSS files, associating a free domain name or subdomain with the server, and securing the site with HTTPS using Certbot. The setup required resolving issues like file listing, permissions, and Apache configurations, showcasing expertise in web hosting, Linux server management, and basic security practices

Also: This project also includes creating an EC2 instance on AWS to serve as the Ubuntu server for hosting the website. The EC2 instance was set up using the Ubuntu operating system, providing a scalable and reliable environment for web hosting.
